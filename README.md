# ⁉️QnA Service


<br>

## 프로젝트 소개
팀프로젝트를 하면서 나의 실력이 부족한것과 다른 기술들도 직접 개발을 해보고싶어서
SpringBoot, JPA를 활용해 QnA 게시판을 만들었습니다.


## 개발 환경
- JAVA 17
- SpringBoot 3.4.0
- JPA
- MySQL
- SpringSecurity
- Git, github


## 주요 기능
- 회원가입/로그인/로그아웃
